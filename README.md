# 🔬 A Multi-Condition Sequential Network Ensemble for Industrial Energy Storage Prediction considering the Condition Switching Characteristics

This repository contains the code implementation of the paper:

 “A Multi-Condition Sequential Network Ensemble for Industrial Energy Storage Prediction considering the Condition Switching Characteristics” 

---

## Project Introduction

Industrial energy storage systems usually operate under multiple production conditions, and the switching behavior between different operating conditions has a significant impact on prediction performance.  
To address this issue, this study proposes a  multi-condition sequential network ensemble framework  for industrial energy storage prediction, with particular emphasis on capturing the  condition switching characteristics  in real industrial processes.

The proposed framework is designed to improve prediction accuracy and robustness under complex, dynamic, and condition-dependent operating environments.

---

## Main Idea

The main motivation of this work is that:

- industrial energy storage data are generated under  multiple operating conditions  rather than a single stationary process;
- the transitions between conditions introduce  distribution shifts  and temporal non-stationarity;
- directly using a single forecasting model often fails to fully characterize the dynamic behavior of the system.

To overcome these challenges, this work develops a sequential ensemble framework that explicitly considers:

-  multi-condition modeling ,  
-  condition switching characteristics , and  
-  sequential prediction of industrial energy storage behavior .

The method aims to better capture the underlying temporal evolution of the system under different operating regimes and improve forecasting performance in practical industrial scenarios.

---

## Code Description

This repository provides the experimental implementation for the proposed industrial energy storage prediction method.  
The code is used to support the model construction, training, testing, and result analysis presented in the paper.

The implementation is developed for research and experimental validation purposes.

---

## Data Availability

The experimental data used in this study are collected from a  real steel plant industrial process .  
Due to  data confidentiality and industrial privacy requirements , the original dataset  cannot be publicly released .

Therefore:

- the raw industrial data are  not included  in this repository;
- only the code implementation and related experimental framework are provided;
- researchers interested in this work may adapt the code to their own industrial datasets or similar benchmark scenarios.

---

## Note

Because the experimental data come from an actual industrial production system, some preprocessing procedures, variable definitions, and engineering details are subject to confidentiality restrictions.  
The repository is intended to present the core implementation idea of the proposed method without disclosing sensitive industrial information.
